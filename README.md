# Overlab - 인터뷰 프로젝트

이 프로젝트는 다양한 인터뷰를 모아놓은 정적 웹사이트입니다.

https://interviewoverlab.netlify.app/

## 📱 프로젝트 소개

Overlab은 다양한 인터뷰를 모아놓은 웹사이트입니다. 각 인터뷰는 독립적인 페이지로 구성되어 있으며, 모바일 환경에 최적화되어 있습니다.

### 인터뷰 목록
- KEEPYOURBLUEMIND 💙
- CHICHEN INTHE LAB 🍗
- 관객의 취향 📚
- 밍밍시 🍀

## 🛠️ 기술 스택

- HTML5
- CSS3
- Pretendard Variable 폰트
- Netlify (배포)

## 🎨 디자인 특징

- 모바일 우선 반응형 디자인
- 깔끔하고 현대적인 UI
- 부드러운 애니메이션 효과
- 최적화된 타이포그래피

## 🚀 배포

이 프로젝트는 Netlify를 통해 배포됩니다.
